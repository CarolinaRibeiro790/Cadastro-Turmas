  Ignite Teams 
</h1>

Este projeto foi desenvolvido durante o curso Ignite Teams da Rocketseat, utilizando as tecnologias React Native e TypeScript. O objetivo principal do aplicativo é gerenciar pessoas em diferentes turmas, permitindo o cadastro e exclusão de participantes.

## 📱 Sobre o Aplicativo
O Ignite Teams é uma aplicação mobile focada em práticas modernas de desenvolvimento React Native, proporcionando funcionalidades como:

<li>Cadastro de Pessoas: Permite adicionar participantes a diferentes turmas.
<li>Gerenciamento de Turmas: Organize os participantes em grupos.
<li>Exclusão de Participantes: Remova pessoas de uma turma de forma prática.
<li>Interface Intuitiva: Design simples e funcional para fácil navegação.
  
## 🚀 Tecnologias Utilizadas
<li>React Native: Framework para desenvolvimento de aplicativos mobile.
<li>TypeScript: Superset do JavaScript para maior segurança e produtividade no código.
<li>Expo: Para simplificar o desenvolvimento, teste e deploy do app.

## 🛠️ Funcionalidades
1. Gerenciamento de Turmas:
* Crie novas turmas para organizar seus participantes.
* Visualize a lista de participantes por turma. <br>
  
2. Cadastro e Exclusão:
* Adicione participantes informando nome e turma.
* Remova participantes individualmente. <br>

3. Persistência de Dados:
* Utilize armazenamento local para salvar os dados das turmas e participantes.
